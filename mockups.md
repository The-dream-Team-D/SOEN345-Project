# Cloud-Based Ticket Reservation Application — UI Mockups

## Overview

This document presents the user interface mockups for the Cloud-based Ticket Reservation Application.  
The goal of the design is to provide a clear, intuitive, and efficient experience for both attendees and event organizers.

The interface focuses on:

- Fast event discovery
- Simple registration flow
- Centralized event management
- Clean and modern visual structure
- Fully responsive layouts

---

## Landing Page

The landing page serves as the main entry point to the platform. It allows users to quickly understand the purpose of the application and start browsing events.

### Main elements

- Hero section with a global search bar
- Featured and upcoming events
- Event categories for quick access
- Call-to-action buttons:
  - Browse events
  - Host an event

### Purpose

The landing page is designed to minimize the time required for users to find relevant events and begin interacting with the platform.

---

## Event Listing Page

The event listing page allows users to explore all available events using filtering and sorting tools.

### Features

- Keyword-based search bar
- Sidebar filters:
  - Category
  - Location
  - Date
  - Price (Free / Paid)
- Sorting options:
  - Date
  - Popularity
  - Price
- Event cards displaying:
  - Event image
  - Title
  - Date and time
  - Location
  - Ticket price
  - Link to event details

### Purpose

This page supports efficient navigation and helps users quickly locate events that match their preferences.

---

## Event Details Page

The event details page provides complete information about a selected event.

### Displayed information

- Event banner
- Event title
- Organizer name
- Date and time
- Location
- Ticket availability
- Event description
- Registration button

### Purpose

This page gives users all the necessary information before registering and provides a clear call to action.

---

## User Dashboard

The user dashboard centralizes all user-related activity and registered events.

### Sections

- List of registered events
- Upcoming and past events
- Digital ticket preview
- Profile management

### Purpose

The dashboard allows users to manage their participation and access their tickets in a single location.

---

## Organizer Dashboard

The organizer dashboard provides tools for creating and managing events.

### Features

- Overview of event statistics
- List of created events
- Registration tracking
- Quick access to event creation

### Purpose

This interface enables organizers to manage their events efficiently and monitor engagement.

---

## Create Event Page

The create event page contains a structured form used by organizers to publish new events.

### Input fields

- Event title
- Event description
- Category
- Date and time
- Location
- Ticket types and quantity
- Price
- Event banner upload

### Purpose

The form is designed to guide the organizer through the event creation process while reducing input errors.

---

## Design System

### Design principles

- Consistent layout and spacing
- Clear visual hierarchy
- Reusable components
- Accessible and readable typography

### Color usage

- Primary color for main actions and highlights
- Secondary color for confirmations and success states
- Neutral tones for backgrounds and containers

### Typography

- Simple and readable font
- Distinct hierarchy for headings and body text

---

## Responsiveness

The interface is designed to work across multiple screen sizes:

- Desktop: full dashboard layout
- Tablet: adapted grid structure
- Mobile: stacked layout with simplified navigation

---

## User Experience Goals

- Reduce the number of steps required to register for an event
- Provide fast and intuitive event discovery
- Offer clear and efficient tools for organizers
- Ensure consistency across all devices
